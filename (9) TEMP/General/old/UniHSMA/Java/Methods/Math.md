# Math.

```java
Math.random()   // will generate a number between 0 and < 1. 
Math.random()   // * n - will generate a number between 0 and < n. 
(int) (Math.random() * n)       // will generate an integer a number between 0 and < n. 
Math.round(Math.random() * n)   // will generate and round a number between 0 and < n. 
```
