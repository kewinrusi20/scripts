## Keywords
```LaTeX
\section{}
\subsection{}
\subsubsection{}

\verb|...| % change text format
```

---
## Example
```LaTeX
\begin{document}

\section{Introduction}
Hello World!

\section{How to print text}
\subsection{...}
\subsection{...}
\subsubsection{}
\section{...}

\end{document}
```

---
# Font
```latex
% Size
\small
\normalsize
\Large
\Huge

% Font
\textbf{} % Bold
\textit{} % Italic

\bfseries % Bold: it works for all the content within a scope
\itshape % Italics: it works for all the content within a scope
```
