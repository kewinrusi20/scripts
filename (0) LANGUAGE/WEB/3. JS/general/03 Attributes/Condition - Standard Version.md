- **Standard** If Statement
```js
if (condition) {
	...
}
```

- **else** Condition
```js
if () {
	...
} else {
	...
}
```

- **Multiple** Conditions
```js
if (condition1) {
    ...
} else if (condition2) {
    ...
} else {
    ...
}
```

- **Nested** (*verschachtelt*) Condition
```js
if () {
	if () {
		...
	}
}
```
