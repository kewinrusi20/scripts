# Ordered List
```html
<h3>Ordered List</h3>
<ol>
	<li>First</li>
	<li>Second</li>
	<li>Third</li>
</ol>
```
- `<ol>` Ordered List
- `<li>` List Element

## Result
```
### Ordered List
1. First
2. Second
3. Third
```

# Unordered List
```html
<h3>Unordered List</h3>
<ul>
	<li>Apple</li>
	<li>Banana</li>
	<li>Cherry</li>
</ul>
```
- `<ul>` Unordered List
- `<li>` List Element

## Result
```
### Unordered List
- Apple
- Banana
- Cherry
```

# Description List
> It's a type of List without dots

```html
<h3>Description List</h3>
<dl>
	<dt>HTML</dt>
		<dd>HyperText Markup Language</dd>
	<dt>CSS</dt>
		<dd>Cascading Style Sheets</dd>
	<dt>JavaScript</dt>
		<dd>Programming language for the Web</dd>
</dl>
```
- `<dl>` Description list
- `<dt>` Term in a description list
- `<dd>` Describes the term in a description list

## Result
```
### Description List
HTML
	HyperText Markup Language
CSS
	Cascading Style Sheets
JavaScript
	Programming language for the Web
```
