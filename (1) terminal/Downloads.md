# Downloads on Msys64
```bash
pacman -S vim
pacman -S make
pacman -S mingw-w64-ucrt-x86_64-gcc # gcc + gpp
pacman -S mingw-w64-x86_64-gdb # gdb "debugger"
```

# Version Program 
```bash
pacman -Suy     # update all
vim --version   # check version
pacman -r       # delete cache
```
